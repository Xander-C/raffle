南邮校科协的寒假项目，算是前端与java组一半一半？前端上传excel表格，后端解析后导入到mysql数据库中，随机从数据库中取出数据放到刮刮卡中。

## 本地安装使用

下载release中的jar包，在命令行中使用` java -jar xxx.jar`运行，需要安装java。

运行后到浏览器中访问` localhost:9091`使用。
